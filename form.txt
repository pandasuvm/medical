Phase 3: Neurological Assessment (GCS)
Glasgow Coma Scale
├── Eye Response (1-4)
├── Verbal Response (1-5) → **CONDITIONAL LOGIC**
│   └── If patient is already intubated → Hide verbal component
│       └── Auto-calculate using: E + M + "T" notation
├── Motor Response (1-6)
│
└── **AUTO-CALCULATION**: Total GCS with interpretation
    ├── 3-8: "Severe brain injury - ICU alert"
    ├── 9-12: "Moderate brain injury"
    ├── 13-15: "Mild brain injury"
    └── **TRIGGER**: If GCS ≤8 → Automatic neuroprotection protocol alerts

Phase 5: Difficult Airway Prediction

├── L - Large Tongue → Points: 0/1
├── E - Thyro-mental Distance → Points: 0/1
├── O - Obstruction → Points: 0/1
├── N - Neck mobility → Points: 0/1

└── **AUTO-CALCULATION & BRANCHING**:
    ├── Score 0-1: Standard intubation approach
    ├── Score 2-3: **TRIGGERS**:
    │   ├── Video laryngoscopy recommendation
    │   ├── Experienced intubator alert
    │   ├── Backup airway devices preparation

        ├── Awake intubation consideration
        ├── ENT immediate consultation
        ├── Emergency cricothyrotomy setup

Phase 6: Real-Time Monitoring Setup
Pre-Induction Baseline
├── Vital Signs Entry
│   ├── Heart Rate → **AUTO-CALC**: Age-adjusted normal ranges
│   ├── Systolic/Diastolic BP → **AUTO-CALC**: MAP, Pulse Pressure
│   ├── SpO2 → **TRIGGER**: If <90% → Preoxygenation protocol
│   └── **AUTO-CALCULATIONS**:
│       ├── Shock Index (HR/SBP)
│       ├── Modified Shock Index
│       └── **RISK STRATIFICATION**: High-risk if SI >0.9

├── Point-of-Care Ultrasound
│   ├── EPSS → **TRIGGER**: If >7mm → LV dysfunction protocol
│   ├── TAPSE → **TRIGGER**: If <17mm → RV dysfunction alert
│   ├── IVC → **TRIGGER**: Fluid responsiveness prediction
│   └── Airway Ultrasound → Difficult airway confirmation

└── Laboratory Integration
    ├── Pre-intubation ABG
    │   ├── pH → **TRIGGER**: Acidosis management
    │   ├── PaO2 → **TRIGGER**: Oxygenation strategy
    │   ├── PaCO2 → **TRIGGER**: Ventilation strategy
    │   └── Lactate → **TRIGGER**: Shock protocol
    │
    └── **AUTOMATED ALERTS**:
        ├── Severe acidosis (pH <7.2) → Bicarbonate consideration
        ├── Severe hypoxia (PaO2 <60) → PEEP optimization
        └── High lactate (>4) → Perfusion monitoring

Phase 7: Medication Selection with Smart Logic
Pre-Induction Medications → **WEIGHT-BASED CALCULATIONS**
├── Fentanyl
│   └── **AUTO-CALC**: 1-3 mcg/kg (based on patient weight)
├── Dexmedetomidine
│   └── **AUTO-CALC**: 0.5-1 mcg/kg loading dose

Induction Agents → **CONDITION-BASED RECOMMENDATIONS**
├── **SMART SELECTION LOGIC**:
│   ├── If Sepsis/Shock → **RECOMMEND**: Etomidate (0.3 mg/kg)
│   ├── If Head Injury → **RECOMMEND**: Propofol (1-2 mg/kg)
│   ├── If Asthma/COPD → **RECOMMEND**: Ketamine (1-2 mg/kg)
│   ├── If Cardiac Disease → **RECOMMEND**: Etomidate
│   └── If Hemodynamically Stable → **RECOMMEND**: Propofol

Paralytic Agents → **TIMING & CONTRAINDICATION CHECKS**
├── Rocuronium (1-1.2 mg/kg)
│   └── **TRIGGER**: If hyperkalemia risk → Warning alert
├── Succinylcholine (1-1.5 mg/kg)
│   └── **CONTRAINDICATION CHECKS**:
│       ├── Hyperkalemia → Block selection
│       ├── Malignant hyperthermia history → Block
│       ├── Neuromuscular disease → Block
│       └── Burns >24 hours → Block

└── **PUSH-DOSE PRESSORS** → **AUTO-PREPARATION**
    ├── If Shock Index >0.9 → Auto-prepare epinephrine/phenylephrine
    └── **DOSAGE CALCULATIONS**: Weight-based ready-to-use doses

Phase 10: Adverse Event Detection & Response
**CONTINUOUS MONITORING FOR ADVERSE EVENTS**
├── **CARDIAC ARREST DETECTION**
│   ├── **TRIGGERS**: HR <60 with hypotension OR asystole
│   ├── **AUTOMATED RESPONSE**:
│   │   ├── CPR protocol activation
│   │   ├── Code blue alert
│   │   ├── Medication preparation (epi, atropine)
│   │   └── Timeline documentation start
│   │
│   └── **FOLLOW-UP FORMS**: Resuscitation documentation

├── **HYPOTENSION MANAGEMENT**
│   ├── **DEFINITION**: SBP <90 OR >20% drop from baseline
│   ├── **AUTOMATED RESPONSE TREE**:
│   │   ├── Fluid bolus recommendation (10-20 ml/kg)

│   │   ├── Vasopressor infusion protocols

│   │
│   └── **ESCALATION TRIGGERS**:
│       ├── Persistent hypotension >10 minutes
